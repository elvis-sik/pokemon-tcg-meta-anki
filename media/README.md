# Media directory

Card and set images are intentionally not redistributed in this handoff.

Run:

```bash
python scripts/download_media.py
python scripts/crop_artwork.py
```

The scripts place generated files here using deterministic filenames. Review
`reports/artwork_crop_contact_sheet.jpg` before building the Anki package.

Pokémon card images, names, and card text remain the property of their
respective rights holders. This repository stores only source locators and
build logic.
